PDFsam

This is a companion utility for the Sourceforge pdfsam project

  • http://sourceforge.net/projects/pdfsam/files/pdfsam

Pdfsam is able to take a PDF file, "BURST" it into individual pages, then "MERGE" individual pages back into a single PDF file.
